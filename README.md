Neural Network with Tanh Activation Function
📌 Overview

This project implements a simple feedforward neural network using the tanh activation function.
The network structure follows the example discussed in class.

The implementation includes:

Random weight initialization in the interval [-0.5, 0.5]

Bias values:

b1 = 0.5 (Hidden layer bias)

b2 = 0.7 (Output layer bias)

Forward pass computation

Printing the final network outputs
